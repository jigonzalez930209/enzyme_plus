# Consejos de validación

- Compara escenarios que difieran por un factor de escalado global (misma `k`). El modelo de salto tau debe mantener formas coherentes.
- Para cantidades y constantes pequeñas, ambos modelos deben producir trayectorias similares (dentro del ruido).
- Supervisa `E + ES + EP` y `S + P + (ES + EP)` para verificar conservación de masa.
